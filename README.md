# utd

Utilities for Instructors at UTD.

## LDAP Example

Perform an LDAP search:

```
$ utd ldap susan

... a nice table of a UTD LDAP search for susan ...

```

## CourseBook Example

Download the PDF photoroster for the named section.

```
$ utd coursebook roster download 18f-math-1314-001.pdf
```
